The plugin provides a widget for receiving virtual currency donations from members of superfluid.

== Description ==

This plugin allows users of superfluid:p2p (http://thesuperfluid.com) to embed a widget that lets other superfluid:p2p users donate virtual currency (with no commercial value) to them via their WordPress blog.
You can customize the width and height of the widget as well as the default donation value. There is also an option to show the widget after every post.

== Installation ==
= Via Wordpress Plugins Directory =
Search for superfluid in the plugins directory and install the plugin by clicking the "install now" link.
= Via Upload =
Simply place the plugin in your plugins directory and activate it in the "Plugins" section of the WordPress administration interface.

After you have activated the plugin, add the widget and configure it. You must at least enter your superfluid id that you can find at http://thesuperfluid.com/my/account.

== Frequently Asked Questions ==

1. Can I use the widget on superfluid:business

     No, this widget and donations in general work only for superfluid:p2p.
